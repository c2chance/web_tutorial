Car Shopee API

This application exposes a small REST API for managing cars. Use the endpoints below to create, read, update and delete car records.

Base URL

http://localhost:8080/api/cars

Endpoints

GET /api/cars

Returns a list of all cars.

Example:

curl -s http://localhost:8080/api/cars

GET /api/cars/{id}

Get a single car by its numeric ID. Returns 404 if not found.

curl -s http://localhost:8080/api/cars/1

POST /api/cars

Create a new car. Provide JSON body with brand, model, and color. Returns the created car with generated id.

curl -s -X POST http://localhost:8080/api/cars \
  -H "Content-Type: application/json" \
  -d '{"brand":"BMW","model":"M3","color":"Blue"}'

PUT /api/cars/{id}

Update an existing car (brand, model, color). Returns 200 with updated car or 404 if not found.

curl -s -X PUT http://localhost:8080/api/cars/1 \
  -H "Content-Type: application/json" \
  -d '{"brand":"Audi","model":"A4","color":"Red"}'

DELETE /api/cars/{id}

Delete a car by ID. Returns 204 on success.

curl -s -X DELETE http://localhost:8080/api/cars/1
